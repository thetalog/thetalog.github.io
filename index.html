todo-app-python

TODO App is a versatile command-line tool for managing lists, designed to handle a variety of tasks with ease. This Python script provides a simple yet effective interface for adding, viewing, deleting, saving, and reading items from a list, with each item assigned a unique identifier. Users can also clear all items in the list when necessary. The script leverages JSON for saving and loading the list to and from a file, ensuring data persistence across sessions. With its intuitive menu-driven approach, Deep's TODO App offers a straightforward solution for keeping track of tasks and managing them efficiently.

pwm

The pwm Password Manager is a comprehensive Bash script designed to securely manage passwords using encryption and decryption mechanisms. The script primarily relies on OpenSSL for generating and handling private and public keys, ensuring secure storage and retrieval of passwords. It includes functions to create directories, check for existing shadow files, and generate unique keys for each password.

rock_paper_scissor_cpp

This C++ program is a simple implementation of the classic "Rock, Paper, Scissors" game where the player competes against the computer (AI). The player selects their throw (Rock, Paper, or Scissors), and the AI randomly generates its throw. The game then compares the throws to determine the winner. If the result is a draw, the game prompts the player to play again until there is a clear winner. The program uses the srand and rand functions to ensure that the AI's throw is randomly selected each time.
